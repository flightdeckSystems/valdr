How this was measured

Loading methodology…

# replay one unit file against the Rust server, exactly as the curve does
$ bash harness/oracle/setup_tcl_runner.sh --skip-build
$ cd reference/valkey
$ VALKEY_BIN_DIR=$(pwd)/../../target/debug \
    tclsh tests/test_helper.tcl --single unit/type/zset \
    --clients 1 --skip-leaks --tags "-needs:repl -needs:debug -external:skip" --quiet
These numbers are scoped to the unit files tracked here, not the full 4,299-block upstream denominator. A file that aborts on a missing global is counted honestly as no-summary (zero), not silently dropped. Read the full accounting in docs/CONFORMANCE.md.
